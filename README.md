# Tarea: Árbol Sintáctico Abstracto (AST) - Compiladores

Este programa genera una representación visual de la estructura jerárquica de dos fragmentos de código: un condicional `if-else` en JavaScript y un bucle `while`.

## 🛠️ Cómo ejecutar el programa

1. Asegúrate de estar en la carpeta principal `TareaCompiladores`.
2. Abre tu terminal y ejecuta:
   ```bash
   python src/main.py
   ```
